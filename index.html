Chris' Pattern Library

Lightweight semantic pattern library that makes things look slightly better than if a frog coded your website.

Typography

Below are the typography styles for this pattern library. Font families used are all web-safe.

Examples:

Heading One

<h1 class="cpl-heading-one cpl-header__title">
  Heading One
</h1>

Heading one is meant to be used for the site title and for the same instances that an h1 would be used.

Heading Two

<h2 class="cpl-heading-two cpl-section__heading">
  Heading Two
</h2>

Heading two is meant to be used as a section heading or even a page heading when the title is abscent.

Heading Three

<h3 class="cpl-heading-three">
  Heading Three
</h3>

Heading three is for separating various divisions in an article.

Heading Four

<h4 class="cpl-heading-four">
  Heading Four
</h4>

In case you need another heading here is heading four.
